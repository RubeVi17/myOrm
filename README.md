# myOrm

myOrm es un ORM (Object-Relational Mapping) ligero y sencillo para PHP que facilita la interacción con bases de datos relacionales mediante una interfaz orientada a objetos. Proporciona una forma intuitiva de definir modelos, realizar consultas y gestionar migraciones de bases de datos.

## Características

- Definición de modelos con propiedades y métodos.
- Consultas sencillas para crear, leer, actualizar y eliminar registros.
- Sistema de migraciones para gestionar cambios en la estructura de la base de datos.
- Soporte para relaciones entre modelos (uno a uno, uno a muchos, muchos a muchos).
- Generación automática de esquemas de base de datos a partir de definiciones de modelos.
- Fácil de extender y personalizar según las necesidades del proyecto.
